# BENLAI

## 项目名称：本来生活（benlailife）

## 项目类型：电商购物类app

> 此应用致力于从国内优质食品供应基地、国外优质食品供应商中精挑细选，剔除中间环节，提供冷链配送、食材食品直送到家服务。我们提供的产品包括：健康安全的蔬菜水果、肉禽蛋奶、米面粮油、母婴童、熟食面点等；具有深厚积淀和历史传承的优质原产地食品；拥有品质保障的进口食品等。
